How to install

The installation of the OpenHRMS software can be done with ease with the help of PostgreSQL and installing it initially and enabling it at the startup. Now you can configure the OpenHRMS software by creating a user, configuring logs, and installing the platform. Finally, configure the server and assign ownership and permission to the user which will finish the entire process of installation and you will be good to run the HR operations.
